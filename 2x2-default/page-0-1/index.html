Схема вращений

  • F - поворот передней стороны по часовой стрелке
  • F' - поворот передней стороны против часовой стрелке
  • B - поворот задней стороны по часовой стрелке
  • B' - поворот задней стороны против часовой стрелке
  • R - поворот правой стороны по часовой стрелке
  • R' - поворот правой стороны против часовой стрелке
  • L - поворот левой стороны по часовой стрелке
  • L' - поворот левой стороны против часовой стрелке
  • U - поворот верхней стороны по часовой стрелке
  • U' - поворот верхней стороны против часовой стрелке
  • D - поворот нижней стороны по часовой стрелке
  • D' - поворот нижней стороны против часовой стрелке

Повороты выполняются только на 90°. Если указано U2, это означает поворот на 180° (два раза по 90°).
