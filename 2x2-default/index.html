Схема вращений

  1. F - поворот передней стороны по часовой стрелке
  2. F' - поворот передней стороны против часовой стрелке
  3. B - поворот задней стороны по часовой стрелке
  4. B' - поворот задней стороны против часовой стрелке
  5. R - поворот правой стороны по часовой стрелке
  6. R' - поворот правой стороны против часовой стрелке
  7. L - поворот левой стороны по часовой стрелке
  8. L' - поворот левой стороны против часовой стрелке
  9. U - поворот верхней стороны по часовой стрелке
  10. U' - поворот верхней стороны против часовой стрелке
  11. D - поворот нижней стороны по часовой стрелке
  12. D' - поворот нижней стороны против часовой стрелке

Повороты выполняются только на 90°. Если указано U2, это означает поворот на 180° (два раза по 90°).
